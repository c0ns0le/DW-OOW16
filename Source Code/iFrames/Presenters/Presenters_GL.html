Overview of Global Leaders Program

The Oracle Data Warehouse (DW) Global Leaders Program is an elite program for key global data warehousing customers. It is managed by the Oracle data warehousing product management team and serves to establish an open communication channel for providing product insights, solution planning and implementation reviews between Oracle Development and customers who are at the leading-edge of data warehousing initiatives. It also provides a rich opportunity for these customers to network with their peers through regional and global communities.
